# Web Scraping Job Market Analysis

This project focuses on collecting job market data through web scraping and performing Data Analysis to uncover insights about job trends, required skills, salaries, and hiring patterns.

---

## 📌 Project Overview

The project involves:

- Scraping job listings data from Internshala
- Cleaning and preprocessing the collected data
- Performing Exploratory Data Analysis (EDA)
- Visualizing hiring trends and insights
- Building an interactive dashboard for analysis

---

## 📂 Files Included

### `internshala_scraped_jobs.ipynb`
Python notebook used for:
- Web scraping
- Data collection
- Data preprocessing

### `Internshala_scraped_jobs.csv`
Dataset containing scraped job listings.

### `Job_Dashboard.ipynb`
Notebook for:
- Data Analysis
- EDA
- Visualizations
- Insights generation
